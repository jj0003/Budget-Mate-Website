Overview

BudgetMate is a sleek Android budgeting app built with Kotlin, targeting SDK API 24 (Nougat, Android 7.0). It simplifies tracking income and expenses, employing Material UI Components 3 for a user-friendly experience. Its minimalistic design makes financial management engaging and efficient.

Features

- Categories Management: Organize transactions into various categories like food, utilities, salary, etc.
- Accounts and Balances: Keep track of your different accounts and their respective balances.
- Expense Tracking: Record and monitor your expenses.
- Transaction Linking: Link transactions to the corresponding categories.
- Data Visualization: View financial data through bar and pie charts for better understanding.

Please note that the app is still in development and that it does not yet work as intended. The app is currently only usable for experimenting purposes and is not yet ready for public use.
